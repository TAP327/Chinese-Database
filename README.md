## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

This is a Chinese dictionary and quizzing device that use a JSON database strongly based on the Chinese Link textbook series. I am updating it, and as I start to do more individualized learning out of class, I will add vocabulary that is not from the textbooks.

<!-- GETTING STARTED -->
## Getting Started

In order to get started, follow the installation instructions below.

### Prerequisites

- Python3
- Pandas
- Tkinter

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:TAP327/Chinese-Database.git
   ```
2. Install Chinese Database App
   ```sh
   pip install -e .
   ```
